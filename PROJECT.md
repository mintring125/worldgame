## ❌ 제외 게임

아래 게임들은 2인용 보드게임이 아니거나, 웹 구현에 적합하지 않아 제외:

- **스포츠류**: 볼링, 골프, 당구, 테니스, 축구, 야구 등
- **1인용 퍼즐**: 슬라이드 퍼즐, 클론다이크, 스파이더 솔리테어
- **다인용 게임**: 루도, 차이니즈 체커, 대부호(프레지던트)
- **성인용/도박성**: 블랙잭, 텍사스 홀덤, 화투, 마작
- **물리 기반**: 다트, 낚시, 사격, 탱크

## 🔧 기술 스택

- **프론트엔드**: HTML, CSS, JavaScript (바닐라)

- **테마**: 동물의 숲 컬러 팔레트
- **서버**: Python HTTP Server (로컬 개발용)
